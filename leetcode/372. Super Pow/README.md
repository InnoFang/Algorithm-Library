Your task is to calculate _a^b_ mod 1337 where _a_ is a positive integer and _b_ is an extremely large positive integer given in the form of an array.

**Example1:**

>a = 2 <br/>
>b = [3]
>
>Result: 8

**Example2:**

>a = 2 <br/>
>b = [1,0]
>
>Result: 1024
